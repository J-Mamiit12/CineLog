About CineLog

A comprehensive JavaScript-powered movie watchlist manager built for ITEL 203 — Web Systems and Technologies.

3 Developers 5 Pages Vanilla JS GitHub Deployed

🎯 Purpose of the System

CineLog is a dynamic, fully client-side web application that empowers users to manage their personal movie watchlist with ease. Users can add films with rich metadata — title, director, genre, release year, and duration — then track their viewing progress, leave star ratings, and search or filter their entire collection. No backend server is required; all data is persisted using the browser's localStorage API, ensuring the collection survives page reloads and browser restarts.

Built as a Group Performance Task for ITEL 203 — Web Systems and Technologies, the project showcases real-world proficiency in DOM manipulation, event-driven programming, strict form validation, algorithmic recommendation logic, and interactive data visualization with Chart.js — all crafted with clean, maintainable vanilla JavaScript.

How the System Works

01

Data Entry & Validation

Users fill out a structured form with movie details. JavaScript listens to real-time input events and enforces strict validation rules — flagging duplicates, checking required fields, and rejecting out-of-range values before any data is accepted.

02

localStorage Persistence

Validated entries are serialized into JSON and stored in the browser's localStorage, acting as a lightweight client-side database. Data survives page refreshes, making CineLog fully functional without any server or internet connection.

03

Dynamic Rendering & Insights

Stored data is read and transformed into dynamic HTML tables, interactive Chart.js visualizations, and a genre-aware recommendations engine — all via DOM manipulation, keeping the UI always in sync with the latest watchlist state.

System Features

Add Movies Input complete metadata: title, director, genre, year, runtime, and status.
5-Star Rating System Interactive star selector to rate any watched film from 1 to 5 stars.
Watch Status Tracking Track each film as Watched, Currently Watching, or Plan to Watch.
Real-time Validation Instant feedback on every field with animated toast alerts for errors.
Dynamic DOM Table Watchlist renders entirely via JavaScript DOM injection — no page reload needed.
Duplicate Detection Prevents adding the same movie title more than once to keep the list clean.
Live Multi-keyword Search Search across title, director, and genre simultaneously as you type.
Category Filtering Quickly filter the full list by watch status with a single click.
Chart.js Analytics Doughnut, bar, and line charts visualise genre, rating, and year data.
Smart Recommendations Algorithm surfaces the best Plan to Watch titles based on your rated genres.
Toast Notifications Animated slide-in toasts confirm every action — add, delete, or update.
Persistent Storage All data lives in localStorage — no login, no server, no data loss.
